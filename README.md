# 产品说明文档测试版

这是一个 GitBook 风格的静态说明文档示例，可以直接用于测试 GitHub Pages。

## 本地预览

直接用浏览器打开：

```text
index.html
```

## 部署到 GitHub Pages

1. 创建一个 GitHub 仓库。
2. 把本目录里的文件推送到仓库。
3. 进入仓库 `Settings`。
4. 打开 `Pages`。
5. Source 选择 `Deploy from a branch`。
6. Branch 选择 `main`，目录选择 `/root`。
7. 保存后等待 GitHub 发布。

发布后的地址通常是：

```text
https://你的用户名.github.io/仓库名/
```
